# Flutter Wrapper

Flutter Wrapper 是一个 VS Code 扩展，拓展了用于快速包裹选中的 Flutter widget 的功能。

## 功能

- 使用 LayoutBuilder 包装选中的 widget

## 使用方法

1. 选中想要包裹的 widget，然后按下 `提取列表`快捷键，选择 `拓展的组件` 即可。
2. 目前支持的组件有：
    - `LayoutBuilder`
    - `Obx`
